# minishell
## A 42 School project

A small, bash-like Linux shell written in C.

## Features

- Simple commands
- Pipes
- Command lists with '&&' and '||' operators
- Simple in/out redirection to files
- Here-doc
- Parameter expansion
- Variable setting/exporting using ``export``
- Builtins:
    - echo
    - cd
    - env
    - export
    - unset
    - exit
    - pwd

## Dependencies

- GNU Readline

## Compilation

``make``
